# SalesSavvyBackend - Spring Boot Backend

## E-Commerce Backend API

A robust Spring Boot backend providing RESTful APIs for the Sales Savvy e-commerce platform with secure authentication and comprehensive business logic.

## Features
- JWT-based Authentication & Authorization
- Role-based Access Control (Admin/Customer)
- RESTful API Design
- MySQL Database Integration
- Payment Gateway Integration
- Order Management System

## Tech Stack
- Java Spring Boot
- Spring Security
- JWT Authentication
- MySQL Database
- Maven
- REST APIs

## Modules
- Authentication Service
- User Management
- Product Catalog
- Shopping Cart
- Order Management
- Payment Processing

## API Documentation
Complete REST API endpoints for all e-commerce operations including user registration, product management, cart operations, and order processing.
